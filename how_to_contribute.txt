Step 1: Fork the repository you want to work on.
Step 2: Clone the repository (git clone URL)
Step 3: Create a new branch to make changes to it (git checkout -b <branch name>) 
Step 4: Make necessary changes in your local repository and add them to the staging area.
Step 5: Commit those changes to your local repository.
Step 6: Push these changes to your GitHub repository (git push origin <branch name>)
Step 7: Create a pull request to the main repository and wait for the request to get accepted!
